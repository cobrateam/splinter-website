test your web applications using Splinter

Splinter is an open source tool for testing web applications using Python. It lets you automate browser actions, such as visiting URLs and interacting with their items.

sample code


from splinter.browser import Browser
browser = Browser()
# Visit URL
url = "http://search.twitter.com"
browser.visit(url)
browser.fill('q', "#cobrateam")
# Find and click the 'search' button
button = browser.find_by_css("#searchButton input").first
# Interact with elements
button.click()
if browser.is_text_present("No results for #cobrateam"):
print "nobody likes us =("
else:
print "we're popular =)"

features

  • simple api
  • multi webdrivers (chrome, firefox, zopetestbrowser)
  • css and xpath selectors
  • support to iframe and alert
  • execute javascript
  • work's with ajax and async javascript
