splinter is a test tool for web applications.

splinter is a tool for test web applications with a simple for find elements, form actions, and others browser actions.

    features

  • simple api
  • multi webdrivers (chrome, firefox, zopetestbrowser)
  • css and xpath selectors
  • support to iframe and alert
  • execute javascript
  • work's with ajax and async javascript
